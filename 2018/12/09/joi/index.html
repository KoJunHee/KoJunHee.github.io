Joi 가 뭡니까 ?

Hapi.js에서 사용하는 parameter 검증 모듈입니다.

Joi 를 사용하지 않으면 ?

routing 의 handler 에서 parameter 를 검증하는 로직과 서버 자원을 조회하는 로직이 섞여있습니다.

Joi 를 사용하면 ?

config 속성을 통해 parameter 를 검증할 수 있습니다. config.validate 속성에 자바스크립트 객체를 설정했습니다.
