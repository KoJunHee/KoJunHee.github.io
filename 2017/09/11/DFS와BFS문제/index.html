1260:DFS와BFS

Sep 11, 2017 in Algorithm

https://www.acmicpc.net/problem/1260

풀이

  • 어레이리스트 생성
    • ArrayList[] a = (ArrayList[]) new ArrayList[n+1];
  • DFS
    • dfs(x) : x를 방문
    • 한 정점에 연결된 정점들 중에
      • 방문하지 않았으면 방문
  • BFS
    • 아직 방문 하지 않았고 현 위치에서 갈 수 있는 정점들을 모두 큐에 넣어
    • 큐에 넣을 때 방문했다고 체크
