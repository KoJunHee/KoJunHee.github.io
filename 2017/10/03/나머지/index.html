1430:나머지

풀이

  1. 그냥 풀면 됨

  2. 이 문제는 풀이가 의미 있는 것이 아니라,
    문제 자체가 의미 있음.
    문제 자체가 알고리즘 풀 때 쓰임.

  3. 다이나믹 문제 풀다가
    int , long long 범위를 넘기 때문에 정답을

    이렇게 구하는 문제가 있음.

    전체 정답을 구하고 나머지 출력하는것이 아님.
